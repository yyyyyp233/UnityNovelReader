# Contributing

Contributions are welcome.

1. Keep the package Editor-only and compatible with Unity 2020.3 or newer.
2. Do not add books, copyrighted samples, telemetry, or project-relative user data.
3. Keep external-reader inspirations at the product-behavior level; do not copy source from differently licensed projects.
4. Add or update EditMode tests for parser, pagination, and persistence changes.
5. Keep public installation and removal instructions accurate.
6. Keep the binary `.unitypackage` to one DLL with embedded license metadata; source and tests belong in the UPM repository, not the imported Unity project.

For security or privacy-sensitive reports, follow the private reporting guidance in [SECURITY.md](.github/SECURITY.md) instead of opening a public issue.

By contributing, you agree that your contribution is licensed under the MIT License.
